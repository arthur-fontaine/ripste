# Installation :

# Prerequisites :

- Docker
- Node.js
- Pnpm

# Steps :

1. Install dependencies:
   ```bash
    npm install
   ```

2. Build the project:
    ```bash
    pnpm build
   ```

3. Start the development server:
   ```bash
    pnpm run start-docker:dev
   ```
